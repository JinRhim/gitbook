# DRV8305 Breadboard Test Board

* 2 layer, JLCpcb&#x20;
* All components should be from leftover components from lab - minimum 0603, TO220 for hand soldering (48HTQFP should not be hard to solder)
* 0.100' pitch connector for breadboard (going to test it with stm32)
